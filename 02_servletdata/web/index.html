공유객체 활용하기

서버에서 활용하는 데이터를 보관하는 객체를 의미. ** 서버는 stateless로 클라이언트와 통신 서버의 서버를 이용한 사용자에 대한 상태관리를 하지 않음!

    공유객체의 종류 3가지 : 데이터 유지 기간의 차이
  • HttpServletRequest : 요청이 생성되고 응답할때까지 데이터가 살아있음
    호출방법 : 매개변수로 들어옴 그냥 이용
  • HttpSession : Session생성하고 Session을 소멸할때까지 데이터가 살아있음
    호출방법 : request.getSession(); -> HttpSession반환
  • ServletContext : server가 실행되고 server가 다운될때까지 데이터가 살아있음
    호출방법 : request.getServletContext() / getServletContext(); -> ServletContext반환

각 공유객체에 값을 넣는 방법 : 공유객체.setAttribute("key(문자열)",value(Object))
각 공유객체에서 값을 가져오는 방법 : 공유객체.getAttribute("key(문자열)") -> 반환값 : Object형이기 때문에 형변환
삭제하는 방법 : removeAttribute("key") / 수정하는 방법 : 동일한 키값으로 setAttribute()
